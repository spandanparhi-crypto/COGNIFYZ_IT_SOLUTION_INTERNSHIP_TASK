Cognifyz Technologies - Data Science Internship
LEVEL 3 - Complete Submission
================================================

Contents:
  task1/  -> Task 1: Predictive Modeling
  task2/  -> Task 2: Customer Preference Analysis
  task3/  -> Task 3: Data Visualization

Each task folder contains:
  - task*.py           : the Python script (fully commented, ready to run)
  - dataset.csv         : the raw restaurant dataset used as input
  - run_log.txt          : full console output from running the script
  - output_*.png         : charts/visualizations generated by the script

HOW TO RUN
----------
1. Make sure Python 3 is installed with: pandas, matplotlib, scikit-learn, numpy
     pip install pandas matplotlib scikit-learn numpy
2. cd into the task folder (e.g. task1/) and run:
     python3 task1_predictive_modeling.py
3. Check the printed console output and the generated output_*.png charts.

Note: task1/ trains Linear Regression, Decision Tree, and Random Forest
models to predict "Aggregate rating" and compares them using MAE, RMSE,
and R2 score. Random Forest achieved the best performance (R2 ~ 0.93).
