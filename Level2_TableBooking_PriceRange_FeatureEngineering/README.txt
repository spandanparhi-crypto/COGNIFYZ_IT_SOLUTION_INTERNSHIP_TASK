Cognifyz Technologies - Data Science Internship
LEVEL 2 - Complete Submission
================================================

Contents:
  task1/  -> Task 1: Table Booking and Online Delivery
  task2/  -> Task 2: Price Range Analysis
  task3/  -> Task 3: Feature Engineering

Each task folder contains:
  - task*.py           : the Python script (fully commented, ready to run)
  - dataset.csv         : the raw restaurant dataset used as input
  - run_log.txt          : full console output from running the script
  - output_*.png         : charts/visualizations generated by the script
                            (task3 instead outputs feature_engineered_dataset.csv)

HOW TO RUN
----------
1. Make sure Python 3 is installed with: pandas, matplotlib
     pip install pandas matplotlib
2. cd into the task folder (e.g. task1/) and run:
     python3 task1_table_booking_online_delivery.py
3. Check the printed console output and the generated output_*.png charts.
